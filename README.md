## Startups
  
- [**`TUNiB (튜닙)`**](http://www.tunib.ai/) : 유명 깃허브 스타이자 자연어처리 리서쳐이신 **박규병**님을 필두로 **카카오브레인** 자연어처리 팀 출신들이 설립한 자연어처리 테크 스타트업. **글로벌 페르소나 챗봇 서비스**를 개발하고 있으며 2021-2022 AI 온라인 경진대회 1위, 그랜드챌린지 우수 기업 선발 등 국내 컴피티션에서의 좋은 성적들과 [**OSLO**](https://github.com/tunib-ai/oslo), [**Parallelformers**](https://github.com/tunib-ai/parallelformers), [**TUNiB-Electra**](https://github.com/tunib-ai/tunib-electra) 등의 오픈소스 기여 등으로 기술력을 인정받고 있으며, 여러  한국어 자연어처리 발전에 기여하고 있다. 대표 서비스로는 한국어 챗봇 서비스 [**Dearmate**](https://dearmate.ai/), 영어 챗봇 서비스 [**Bloony**](https://bloony.ai/)와 자연어처리 API 서비스 [**TUNiBridge**](https://tunibridge.ai/) 등이 있다. 21년 하반기에 **30억 규모의 시드 투자**를 마무리했다. ***(Last Update: 2023.01.26)***
